Conociendo el sistema

Cuando hayas iniciado sesión, estarás dentro del sistema donde podrás ver la barra de navegación o el menú el cual tiene algunas funciones, el rol y el correo, los componentes como el tanque, la válvula, batería, las funciones de la corriente que activa los componentes y a su vez el de stop que desactiva estos componentes y por último las líneas de riego con sus funciones correspondientes como es activar la línea y ver las estadísticas tanto de humedad como de temperatura.

Activar los componentes

Los componentes son aquellos que hacen parte del sistema, estos proveen información concisa que ayuda al funcionamiento de las líneas. Se pueden activar estos componentes precionando el botón azul.

Desactivar los componentes

Para desactivar los componentes se preciona el botón rojo.

Líneas de riego

Las líneas de riego tienen datos que son el número de la línea correspondiente, el estado, la temperatura y la humedad.
También tienen funcionalidades como activar la línea y ver las estadísticas tanto de humedad como de temperatura.

Funciones de las líneas de riego

Botón activar/desactivar
El botón Activar que corresponde a cada línea hace que el estado cambie de Apagado a Activado.
A su vez, cuando se acciona ese botón, cambia también de estado Activar a un estado Desactivar
Botón ver humedad
El botón E. humedad te redirecciona a una sección del sistema que es para ver las estadísticas de la línea siendo la humedad el dato a ver.
(Funciona si la línea está activada).
Botón ver temperatura
El botón E. temperatura te redirecciona a una sección del sistema que es para ver las estadísticas de la línea siendo la temperatura el dato a ver.
(Funciona si la línea está activada).

Estadísticas humedad

Para ver las estadísticas debes activar primero la línea a la que quieres acceder a esa función.
Activar cada línea te da acceso tanto a sus datos como estado, temperatura y humedad como también acceso a ver las estadísticas de humedad y temperatura.

Funciones

Hay cinco funciones en esta interfaz las cuales son:

Tipo de gráfica

Tiempo real

Los datos se irán actualizando constantemente.

Descargar informe

Descargar gráfica

Podrás descargar la gráfica en formato de imagen.

Volver

Puedes regresar al inicio con las demás líneas.

Gestionar sensores

Tipo de sensor

Hay dos tipos de sensores, está el de humedad y el de temperatura, los dos sensores tienen las mismas funcionalidades pero para diferente fin. Cuando le hayas dado clic al enalce Tipo de Sensor, allí se desplegará una lista que contiene los dos tipos de sensores así como se muestra en la imagen.

Acciones

Estas acciones son compartidas tanto para el sensor de humedad como para el sensor de temperatura.

Crear

Puedes crear un nuevo sensor, dependiendo en qué sección de la interfaz estés, puedes crear uno nuevo.

Editar

Puedes editar el sensor que quieras, partiendo de que puedes cambiar el nombre o la planta a la que pertenece.

Eliminar

Puedes eliminar el sensor que quieras teniendo en cuenta que ya no se va a usar o que fue un error haberlo creado.

Informe en Excel

También puedes descargar un informe en Excel mostrándote los registros en las celdas de Excel.

Listar

Puedes elegir la cantidad de sensores a mostrar en la interfaz partiendo de la base hasta 100 registros.

Buscar

Si estás desubicado, puedes buscar un sensor en particular en el campo buscar.

Permisos para acceder a las plantas

En esta interfaz no necesitas ser administrador, puedes ingresar con el rol de jardinero sin problema.

Interfaz plantas

Esta es la interfaz de las plantas, donde se puede crear, listar, editar y eliminar la variedad de plantas que puedan haber en un vivero.

Editar

Eliminar

Si no te aparece de esta manera, posiblemente estés en una resolución de pantalla inferior a la de un computador o laptop. Para este caso, en la parte superior derecha, hay un icono en diseño de lista así como se muestra en la siguiente imagen.
Una vez dado el clic sobre el icono, se desplegará una lista, identifica que en la lista esté tu correo con el que te registraste así como se muestra en la siguiente imagen.
